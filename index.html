Our Motivation

Notional Trace is a web application designed to help novice programmers learn how computers execute particular pieces of code. Often times, students struggle with introductory programming courses (whether it be at the middle school, high school, or college level). Much of the difficulty comes down to not having a mental model of how a piece of code produces desired output.

Our goal with Notional Trace is to help students build a notional machine. A notional machine is basically just a mental model of program execution from the computer’s standpoint. Having a strong notional machine will enable students to become faster, more efficient programmers, and will make coding a more enjoyable practice.

Notional Trace utilizes code tracing to help students build their respective notional machines. Various studies have been conducted proving the effectiveness that code tracing has on introductory programming concepts, therefore that’s the route we will take within our learning modules. Ultimately, we hope to help students become better programmers, and ultimately broaden participation in computing.

Modules

Select a module and start learning!

Variables and Datatypes

Create variables in order to store various different types of data. Use these variables to perform specific operations and write functional programs.

Selection Statements

Want to only execute a block of code if a certain condition is met? Selection statements allow you to do this and build more powerful programs.

Loops

Loops allows programmers to execute a block of code several times. This is an extremely important concept as it saves programming time, and shortens code!

Arrays

So you know about variables, however what if you want to make a collection of them? Arrays make it easy to create a bunch of variables of the same data type!

Stacks

Think of a stack of plates. This data structure allows for two main functions: pushing elements onto the structure, and popping them off.

Queues

Conceptually similar to a stack, however a queue is essentially just a row of variables. Queues also have two main functions: enqueue and dequeue.
