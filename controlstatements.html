Control Statements

Introduction

Loops become useful when we want to execute a block of code multiple times. Loops allow code blocks to repeat as long as a specific condition is met. This is a very powerful statements that allow us to shorten the amount of code we have to write, ultimately saving programmer time. There are many different types of loops that are appropriate in different situations, however we are going to look at the two most popular kinds of loops: the while loop and the for loop.
